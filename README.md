## Usage

Create file with all private data to connect ToDoist and Discord bot.

`.env`
```
{
db_root_user=...
db_root_pass=...
todoist_client_id=...
todoist_client_secret=...
todoist_scope=...
discord_token=...
logging_file=...
}
```
